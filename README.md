# Portfolio - BUT Réseaux et Télécommunications

Bienvenue ! Voici votre portfolio HTML5/CSS3 prêt à être personnalisé.

## 📁 Structure des fichiers

```
index.html      - Page principale (structure HTML)
style.css       - Feuille de style (design et mise en page)
script.js       - Interactivité (menu mobile, smooth scroll)
profile.jpg     - Photo profil (à remplacer par votre photo)
cv.pdf          - Votre CV (à ajouter)
```

## 🎨 Personnalisation

### 1. **Informations personnelles**

Ouvrez `index.html` et modifiez :
- Votre nom et titre en haut du fichier
- Votre description d'accueil (section hero)
- Vos coordonnées (email, LinkedIn, GitHub)
- Les sections "À propos", "Projets", "Compétences", "Expériences"

### 2. **Photo profil**

- Remplacez `profile.jpg` par votre photo (format JPG, PNG)
- Dimensions recommandées : 400x400px ou plus
- Assurez-vous que la photo est de bonne qualité

### 3. **Projets**

Complétez les 3 projets selon le PDF :
- **Projet 1** : Projet technique (ex: Administration réseau)
- **Projet 2** : Deuxième projet technique
- **Projet 3** : Projet non-technique

Structure de chaque projet :
- Nom du projet
- Objectif/problème
- Actions réalisées (liste à puces)
- Technologies utilisées
- Lien vers les preuves (documents, GitHub, etc.)

### 4. **Compétences**

Les 4 catégories sont :
- 🔧 Réseaux
- 🔐 Sécurité
- 💻 Développement
- 🛠️ Outils

Adaptez les listes selon vos vraies compétences.

### 5. **Expériences**

Ajoutez votre timeline professionnelle :
- Dates
- Titre du poste/formation
- Description

### 6. **Design & Couleurs**

Les couleurs utilisées (à modifier dans `style.css` si souhaité) :
- Bleu (#2563eb) - Couleur primaire
- Bleu foncé (#1e40af) - Variante
- Orange (#f59e0b) - Accent (CTAs)
- Gris foncé (#1f2937) - Texte

Pour changer les couleurs, modifiez les variables CSS en haut de `style.css` :
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #f59e0b;
    /* etc... */
}
```

## 📱 Responsive

Le portfolio s'adapte automatiquement à tous les appareils :
- Desktop (1200px+)
- Tablette (768px - 1199px)
- Mobile (moins de 768px)

Un menu hamburger s'affiche sur mobile.

## 🚀 Déploiement

Pour mettre en ligne votre portfolio :

1. **Netlify** (gratuit, simple) : Drag & drop le dossier
2. **GitHub Pages** : Push sur GitHub, activez Pages
3. **Vercel** : Connectez votre repo GitHub
4. **Heroku** : Déployez avec git push

## 💡 Conseils

1. **Cohérence graphique** : Gardez les mêmes couleurs, polices et style d'images
2. **Preuves visuelles** : Ajoutez des schémas, screenshots, liens GitHub
3. **Qualité avant quantité** : 3 bons projets > 10 petits projets
4. **Mise à jour** : Complétez cette année, enrichissez-le jusqu'à la fin du BUT
5. **Testez** : Ouvrez le portfolio sur différents appareils (mobile, desktop)
